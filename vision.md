# Vision — FitnessTracker

## Problem
Users need a simple way to plan, track, and review their workout sessions without relying on cloud services.

## Solution
A console-based fitness tracking application that allows users to create workouts, log exercises, and review training history — all stored locally.

## Target Users
Individual athletes and gym-goers who want local, offline workout logging.

## Success Criteria
- User can create a workout and add exercises in under 30 seconds
- History is persisted between sessions
- All error states produce clear messages
